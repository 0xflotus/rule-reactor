Send More Money tests a rule engine's ability to handle large numbers of possible combinations of values (i.e. 10^8 combinations, 100,000,000) to solve a crypto-mathematical problem. Nools takes approximately 1.5 minutes in Firefox on an Intel i7-4510U 2.00GHz with 8GB RAM and Windows 10 Pro 64. Under the same configuration, RuleReactor will solve the problem in approximately 2 minutes. We think this is pretty good for directly coded JavaScript rules that can be handled in the regular debugger and an engine that is only 24K (11K minified) vs 577K (227K minified) for Nools (granted Nools does have some additional features). The solution takes approximately 3 minutes in Goggle Chrome. It takes substantially longer in Microsoft Edge. Your browser will probably appear to hang while running this example. We are working to resolve this. For now, be patient.
